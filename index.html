Welcome to my GSM cell phone unlocking guide!

The following information pertains to ALL gsm cell phones.

  • Always charge the cell phone before servicing.
  • The database may not always have the correct IMEI for the phone ID that you're servicing. Be sure to get the IMEI from the phone itself.
  • All unlock boxes, dongles, clips, etc are located at the top right drawer of the unlocking desk. Always put the unlocking box back in its location after use.
  • When connecting the unlocking boxes to the computer, always use the USB-B cable that is located above the keyboard (unless otherwise specified).
  • A microUSB and miniUSB cable are already plugged in and will be used for nearly 98% of all unlocking operations.
  • Never disconnect or handle the phone while the unlocking software is doing it's task. Doing so may brick the device and render it useless.
  • Do not use the computer heavily while a cell phone is in the process of being unlocked. A hang up of the software due to high CPU usage may break the device.
  • It is much faster to group the cell phones by manufacture when unlocking. This eliminates the need to remove and plug in different unlocking boxes.
  • Periodic maintenance of the unlocking boxes is required. This should only be handled by Mathew. This includes, but not limited to, software updates, box firmware updates, credit(s) purchasing.
  • Always service FedEx Express shipments as early as possible. If there is an issue with the unlock, you will be able to address this sooner
  • In the event that you damage the device, please consult Mathew as repair is out of the scope of this document. Attempting to fix a device yourself may cause further damage.

Apple

Unlocking Instructions

Using IMEIdata.com

  • 1. Go to imeidata.com
  • 2. Click log in at the top right if not already signed in, user details are already saved
  • 3. Mouse over IMEI services and select Place IMEI Order
  • 4. In Service drop down, select 12 – iPhone AT&T Unlock(3,3gs,….)[Clean & Out of contract)
  • 5. In Process type*:, select Instant (submit 1-by-1)
  • 6. Enter the CORRECT IMEI in the BULK IMEI field/box
  • 7. In note and comment boxes, enter in the Phone ID and Order Number
  • 8. Click Place Order
  • 9. Verify that the order went through by mouse over IMEI services, and clicking Order History

Unlocking Instructions

Using AllUnlockGSM.com

  • 1. Go to allunlockgsm.com/index.php
  • 2. Click Login and push click the login button, user details are already saved
  • 3. Mouse over Place Order and select IMEI Service
  • 4. In services drop down, select AT&T Fast (Clean) – iPhone 4s |… (instant – 24hr)
  • 5. Enter the CORRECT IMEI in the BULK IMEIE field/box
  • 6. In comment box, enter in the Phone ID and Order Number
  • 7. In response e-mail, type in mattm@cellularcountry.com
  • 8. Click Place Order
  • 9. Verify that the order went through by clicking the Dashboard link just below the navigation bar

Samsung

Samsung Android Newer Generation (Preferred Method) Models

Samsung Galaxy Express (SGH-I437P), Samsung Galaxy Alpha (SM-G850A), Samsung Galaxy Rugby Pro (SGH-I547), Samsung Galaxy S5 (SM-G900H)

Preferred Method Instructions

  • 1. Open Chimera Tool on desktop
  • 2. Dial *#0808#
  • 3. Select DM+MODEM+ADB, push ok
  • 3.a. If the phone is Android Version >= 5.0, Turn the phone off. Hold home key+ volume up. Select Enable CP Logging
  • 4. Connect Phone to PC, let drivers install
  • 5. Phone may ask for 'Allow USB Debugging', check mark 'Always allow..', and click ok
  • 6. Click Unlock button in Chimera Tool
  • 7. If successful, reboot the phone if it didn't do it by itself. Dial *#0808#, select MTP+ADB, click ok. Turn the phone off

Samsung Android Newer Generation (Secondary Method) Models

Almost all models as Preferred method, Samsung Galaxy S4 Active (SGH-I537)

Secondary Method Instructions

  • 1. Connect Z3X Samsung Box to USB plug above keyboard
  • 2. Open Samsung Tool PRO on desktop
  • 3. Select the CORRECT phone model on the left side of the software.
  • 4. Dial *#0808#
  • 5. Select DM+MODEM+ADB, push ok
  • 5.a. If the phone is Android Version >= 5.0, Turn the phone off. Hold home key+ volume up. Select Enable CP Logging
  • 6. Connect Phone to PC, let drivers install
  • 7. Phone may ask for Allow USB Debugging, check mark 'Always allow..', and click ok
  • 8. Go to the Unlock tab at the top of the software window
  • 9. Select the highest Samsung Mobile USB Serial Port in the Com drop down menu.
  • 10. Click the Unlock button on the right, If unlock fails, try the other Samsung Mobile USB Serial Port if there is one
  • 11. If successful, reboot the phone if it didn't do it by itself. Dial *#0808#, select MTP+ADB, click ok. Turn the phone off

Samsung Android Older Generation Models

Samsung Galaxy Vibrant (SGH-T959)*, Samsaung Galaxy S 4G (SGH-T959V)*, Samsung Galaxy S2 (SGH-I777)*
*Models that need to be rooted with other software*

Older Generation Instructions

  • 1. Turn on ADB Debugging
  • 2. Connect Z3X Samsung Box to USB plug above keyboard
  • 3. Open Samsung Tool PRO on desktop
  • 4. Select the CORRECT phone model on the left side of the software.
  • 5. Click on the Unlock tab
  • 6. Select a device (should only have one) in the adb drop down
  • 7. Click unlock
  • 8. Once complete allow the phone to idle for a minute; dial *#06#; If the imei is damaged, you will need to do an imei repair procedure
  • 9. Check unlock is complete by dialing *#7465625#

Samsung Android Older Generation Models
